Intro to Terminal

Today, I'll be teaching you to navigate the terminal. The terminal is a tool that lets you communicate with your computer using just text commands. There are no menus or confusing visuals, but rather commands such as cd, ls, etc...

Why Should YOU Learn the Terminal?

  • It's faster for things such as file editing
  • It grants access to tools that you couldn't access otherwise
  • It shows you how your computer works on a deeper level!

Starter Guide for Terminal Commands

These are some of the commands that'll help you navigate the terminal on your first visit:

  • cd - changes which directory you're in (ex. cd desktop takes you to your desktop.)
  • ls - lists what files are in the current directory (ex. ls (while in desktop) will show you what folders are on your desktop)
  • pwd - shows you where you are on your computer (ex. pwd while you're in your desktop will say you're in your desktop)
  • mkdir - makes a new directory (ex. mkdir cool_stuff makes a new directory called cool_stuff)

Now that you've learned the basics, let's give an example of how to make a folder called "folder_1" in the terminal!

  1. Open the Terminal.
  2. Type pwd to check where you are on the computer
  3. Type cd desktop to move to your desktop
  4. Type mkdir folder_1 to make a folder called "folder_1"
  5. Type cd folder_1 to go inside the folder
  6. Type pwd to make sure you're inside the right folder
  7. Enjoy the new folder!

Here's an example session you might see when running commands:


		% pwd
		/student/Desktop
		% ls
		folder_1 folder_2 folder_3
		% cd folder_1
		% pwd
		/student/Desktop/folder_1

How to Enter Vim

Let's teach you how to edit a file with Vim.

Vim is a program where you can insert text within a file and turn it into a website, informative presentation such as this one you're reading, or more.

Let's teach you how to use Vim to edit a file called "teachmevim.html" inside the "folder_1" directory.


		% pwd
		/student
		% cd desktop/folder_1
		% pwd
		/student/Desktop/folder_1
		% vim teachmevim.html

Once you complete the steps above, you should be in Vim to edit your file called "teachmevim.html"

Vim Commands & How to Use Them

Now that you know how to enter Vim, how do you EDIT a file

Well here's a few simple commands to get you started:

  • Type I to enter insert mode, which allows you to enter text.
  • When inside of insert mode, you can type esc followed by :syntax on to highlight the code in colors to make it easier to read.
  • Once you're finished editing in Vim, you can type esc followed by :wq to save and exit the file with all your data remaining within.

Conclusion

Though the terminal may feel intimidating at first, practice makes perfect when it comes to using and mastering it. Just keep practicing, and you'll become a pro at it.
